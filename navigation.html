The vertical navigation bar was created by adding an id to the "nav" tab. Then using css values I eliminated the list by using the "list-style" attribute under the "ul" tag. The color pattern I used was created by using the color picker from "W3Schools" and I chose a fuchsia gradient by picking every other shade of the color. I started with 50% and worked my way down to 90%. The font I chose was from Google Fonts, I had to add the link of where I got the fonts and had to import the URL to the CSS file. Vertical navigation menu would be beneficial if there were longer worded links or if there was going to be added links in the future. Having the navigation vertical allows for easy upgrades without having to change much.
